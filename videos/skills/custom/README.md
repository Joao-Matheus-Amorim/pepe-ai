# Skill: custom

Pasta para skills personalizadas. Coloque vídeos sobre qualquer assunto que queira ensinar ao Pepê.

## Formatos suportados
`.mp4`, `.mp3`, `.wav`, `.m4a`, `.mkv`, `.avi`, `.mov`, `.webm`

## Como extrair
```bash
python training/extract_skills.py --skill custom
```

## Criar uma nova skill
```bash
python training/skill_manager.py new minha_skill --description "Descrição da skill"
```
